# 📄 Resume App — Fullstack CV Application

Moderná fullstack aplikácia môjho osobného životopisu vytvorená s dôrazom na čistý dizajn, viacjazyčnú podporu a responzívne používateľské rozhranie.

Frontend aj backend sú kompletne oddelené a deploynuté samostatne pre lepšiu škálovateľnosť a správu projektu.

---

## 🌍 Live Demo

👉 [Resume App Live Demo](https://resume-app-vert-two.vercel.app/)

---

# ✨ Features

- 🌐 Prepínanie jazykov jedným kliknutím
- 📄 Dynamický životopis s prekladmi
- 📱 Responzívny dizajn pre mobil, tablet aj desktop
- ⚡ Rýchly frontend postavený na Preact + Vite
- 🔒 Oddelený backend API
- 🎯 Moderný TypeScript stack
- 🚀 Deploy:
  - Frontend → Vercel
  - Backend → Render

---

# 🛠️ Tech Stack

## Frontend

## Backend

- TypeScript

# 📱 Responsive Design

Aplikácia je plne responzívna a optimalizovaná pre:

- 📱 Mobile
- 💻 Desktop
- 📟 Tablet

Používateľské rozhranie sa automaticky prispôsobuje veľkosti obrazovky pre čo najlepší používateľský zážitok.

---

# 🌍 Multilanguage Support

Projekt obsahuje implementované preklady a možnosť okamžite meniť jazyk pomocou tlačidla v používateľskom rozhraní.

To umožňuje jednoduché prepínanie medzi jazykmi bez potreby reloadu stránky.
